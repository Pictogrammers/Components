# `<pg-input-select>`

The `pg-input-select` component creates an select drop down.

```typescript
import '@pictogrammers/components/pgInputSelect';
import { PgInputSelect } '@pictogrammers/components/pgInputSelect';
```

```html
<pg-input-select></pg-input-select>
```

## Attributes

| ---------- | -------- | ----------- |

| value      |          | `world` |
