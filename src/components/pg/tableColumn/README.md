# `PgTableColumn`

See `PgTable`. Default column header.

| Attributes | Tested   | Default | Description |
| ---------- | -------- | ------- | ----------- |
| label      |          | `''`    | Header text. |
| hideLabel  |          | `false` | Render empty with `aria-label` set to `label`. |
| key        |          | `''`    | Column key. |

| CSS Variable        | Default   | Description |
| ------------------- | --------- | ----------- |
| `--pg-table-border-color` | `#453C4F` | Bottom border |
