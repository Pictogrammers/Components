# `<pg-markdown>`

Render markdown text with rich formatting.

```typescript
import '@pictogrammers/components/pg/markdown';
```

## Usage

All markdown formatting is supported. Code blocks support:

- css
- groovy
- javascript
- json
- jsx
- tsx
- java
- markup (aka html, xml)
- typescript
- sass
- scss
- yaml (with a custom viewer)
- php
- bash
- markdown
