# `PgInputPill`

The `PgInputPill` is used internally as items are added to the `pg-input-pills`.

## Attributes

| Attributes | Tested   | Description |
| ---------- | -------- | ----------- |
| label      |          | Pill label text. |
| value      |          | Pill value. |

## Events

| Events     | Tested   | Description |
| ---------- | -------- | ----------- |
| remove     |          | Remove button clicked. `{ detail: { value, label } }` |

## CSS Variables

| CSS Variables       | Default   | Description |
| ------------------- | --------- | ----------- |
| `--pg-input-pill-background-color` | `#EDEAF1` | Pill background color. |
| `--pg-input-pill-color` | `#453C4F` | Pill text color. |
| `--pg-input-pill-border-radius` | `1rem` | Pill border radius. |
| `--pg-input-pill-remove-hover-background-color` | `#453C4F` | Remove button hover background. |
| `--pg-input-pill-remove-hover-color` | `#FFF` | Remove button hover color. |
| `--pg-input-pill-remove-icon-size` | `0.875rem` | Remove icon width / height. |
